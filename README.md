# MusicPlayerApp

# About
An app that fetches music from the iTunes API. Functionalities include searching and playing music.

# Pre-requisites
* Xcode 13 beta (optional)

# How to launch
* Open Xcode
* Tap on play button 
* App should launch in the simulator
